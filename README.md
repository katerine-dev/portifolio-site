# portifolio-site

This project is an exercise to learn front-end development.


Enabling virtual environment:
```sh
poetry shell
```
Installing a single depdency:
```sh
poetry add flask
```
The line above adds the `requests` library

Installing project dependencies:
```sh
poetry install
```
